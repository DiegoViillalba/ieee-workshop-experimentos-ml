1 Acerca de este proyecto

Este sitio documenta el experimento completo desarrollado durante el Workshop “De la idea al experimento” (Días 1–4).

Partimos de una pregunta científica, construimos un pipeline reproducible, entrenamos y validamos dos modelos, y comunicamos los resultados en múltiples formatos desde un único documento fuente.

2 Pregunta científica

¿Un modelo de regresión logística con ponderación de clases alcanza recall ≥ 0.90 para la clase maligna y supera a un clasificador de mayoría en AUC-ROC?

Resultado: H₁ sostenida. Recall = 0.976, AUC-ROC = 0.994.

# 1. Renderizar (output va a docs/)
quarto render

# 2. Subir a GitHub
git add docs/
